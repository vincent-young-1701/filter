# Filter

Does a bit of filtering.

## Installation

Use NPM and get all the codes.

```bash
npm install
```

## Running

Run in development

```bash
npm run start
```

Build for production

```bash
npm run build
```

Test

```bash
npm run test
```

## Features

- Built with latest React (Hooks, Context API, etc.)
- Search by URL (?search=keyword)
- Fully accessible
